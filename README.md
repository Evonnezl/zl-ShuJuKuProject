# 房产管理系统

## 项目简介

本项目为数据库课程设计——**房产管理系统**。

系统主要用于房屋资源管理和用户住房申请管理，支持 **分房、调房、退房和咨询统计功能**。

房产科将用户提交的申请表输入系统后，系统会先检查申请表合法性，并对合法申请进行自动处理。

---

## 功能需求

### 分房申请

- 根据申请者条件计算分数  
- 分数高于阈值则进入分房队列（按分数排序）  
- 每月统一进行分房：
  - 从空房列表获取房源
  - 按分数优先级分配房屋
  - 更新住房信息
  - 输出分配结果并计算房租

---

### 退房申请

- 删除住房记录及相关信息
- 将房屋状态重新标记为空房
- 回写空房列表

---

### 调房申请

- 根据申请者条件确定住房等级
- 查找符合条件的空房
- 释放原住房并重新分配

---

### 咨询与统计

- 查询分房阈值分数
- 查询不同房屋类型条件
- 查询房屋单位面积租金
- 输出住房统计表
- 支持修改房屋标准与租金

---

## 数据库设计

### 表结构说明

| 表名               | 主要字段                                 | 说明           |
| ---------------- | ------------------------------------ | ------------ |
| user             | id, name, role, phone, email         | 用户信息（住户/管理员） |
| house            | id, title, status, area, rent_per_m2 | 房源信息         |
| application      | 申请信息                                 | 分房/调房/退房申请记录 |
| housing_standard | area, min_score                      | 住房标准         |
| housing_record   | user_id, house_id, score             | 已分配住房记录      |
